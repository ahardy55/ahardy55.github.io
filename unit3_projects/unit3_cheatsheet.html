Data Types

String Holds a sequence of characters string = "I am a string"
Boolean A true/false value boolean = true
Array A range of values array = [1,2,3,4,5]
Hash A collection of key/value pairs hash = {
  "Marky" => 35,
  "Sarah" => 44,
  "Jenny" => 22
}

Looping

.each Looks at each element array.each do |element|
   puts element
end
until Loops until condition is met until i == 5
   puts "Still going"
   i += 1
end
while Loops while condition is true money = 500
while money > 0
   puts "You still have money"
end
times Set number of times loop will run 5.times do
   puts "I'm running 5 times"
end
for Executes for each element for element in array do
   puts element
end

Common Methods

.each Calls the given block once for each element Returns each item
.include? Checks if given object is present Return true or false
.length Used to find number of elements Returns the number of elements
.map Call given block and creates new array from modified values returns new array
.sort sorts elements returns sorted list
